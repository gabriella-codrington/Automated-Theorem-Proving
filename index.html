Overview

Automated Theorem Proving (ATP) deals with the development of computer pro- grams that show a statement is a logical consequence of a set of axioms. A notable theorem prover is the Boyer-Moore theorem prover (NQTHM) developed by Robert S. Boyer and J. Strother Moore using Pure LISP [4]. The prover forms definitions as totally recursive functions and makes extensive use of rewriting and an induction heuristic that is used when rewriting and symbolic evaluation fails. Boyer-Moore was used to confirm Kurt G ̈odel’s First Incompleteness Theorem which states the following: any consistent formal system F within which a certain amount of elemen- tary arithmetic can be carried out is incomplete; i.e., there are statements of the language of F which can neither be proved nor disproved in F. The Boyer-Moore theorem prover takes a series of lemmas are input and the system generates the proofs. However, using the Boyer-Moore proof system requires feeding it a “well- chosen sequence of lemmas”, so it would seem the information being fed into the two systems is similar.
